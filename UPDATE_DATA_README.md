# سكريبت تحديث البيانات الواقعية

## 📋 الوصف
سكريبت TypeScript/Drizzle ORM لتحديث البيانات الحالية في قاعدة البيانات لتصبح منطقية وواقعية.

## 🎯 ما يفعله السكريبت

### 1. تحديث بيانات المرضى
- ✅ تغيير الأسماء إلى أسماء عربية منطقية
- ✅ تحديث أرقام الهواتف إلى أرقام مصرية صالحة (+20XXXXXXXXXX)
- ✅ تعديل البريد الإلكتروني ليكون متناسقًا (first.last@test.com)
- ✅ الحفاظ على معرفات UUID الحالية

### 2. تحديث بيانات الأطباء
- ✅ تعديل الأسماء إلى أسماء عربية أو معروفة
- ✅ تعديل البريد الإلكتروني ليكون متناسقًا
- ✅ تحديث التخصصات الطبية (general, dentist, pediatric, etc.)
- ملاحظة: جدول الأطباء لا يحتوي على حقل رقم الهاتف في النسخة الحالية

### 3. تحديث أوقات العمل
- ✅ إنشاء جدول عمل واقعي للأطباء
- ✅ الأحد - الخميس: 9:00 ص - 5:00 م
- ✅ الخميس: 9:00 ص - 3:00 م (ساعات أقل)
- ✅ الجمعة والسبت: عطلة

### 4. تحديث المواعيد
- ✅ ربط المواعيد بالمرضى والأطباء المحدثين
- ✅ تعديل التواريخ لتكون منطقية
- ✅ تحديث الحالات (pending, confirmed, cancelled, done)
- ✅ تحديث الأولويات (normal, priority, emergency)

### 5. تحديث قائمة الانتظار
- ✅ ربط السجلات بالأطباء المناسبين
- ✅ تحديث بيانات المرضى في قائمة الانتظار
- ✅ تعديل التواريخ المفضلة

## 🚀 كيفية التشغيل

### الطريقة 1: باستخدام npm script
```bash
npm run db:update-data
```

### الطريقة 2: مباشرة باستخدام tsx
```bash
npx tsx update-realistic-data.ts
```

## ⚠️ ملاحظات هامة

1. **نسخ احتياطي**: قبل تشغيل السكريبت، يفضل أخذ نسخة احتياطية من قاعدة البيانات
2. **UUIDs**: يتم الحفاظ على جميع معرفات UUID الحالية لتجنب كسر العلاقات
3. **بيانات واقعية**: البيانات الجديدة مصممة لتكون واقعية ومناسبة للاختبار
4. **التشغيل مرة واحدة**: السكريبت مصمم للتشغيل مرة واحدة فقط
