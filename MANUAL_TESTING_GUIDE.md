# دليل الاختبار اليدوي الشامل لجميع الأدوار

**التاريخ:** 2026-04-23
**النوع:** اختبار يدوي شامل
**الهدف:** اختبار جميع الأدوار والسيناريوهات وحل مشاكل تجربة المستخدم

---

## 📋 الإعداد

### 1. تشغيل التطبيق
```bash
npm run dev
```
التطبيق سيعمل على `http://localhost:3000`

### 2. بيانات الدخول
- **سوبر أدمن:** admin@platform.com / Admin@1234
- **مدير العيادة:** clinic@dr-ahmed.com / Clinic123!
- **الموظفين والأطباء:** سيتم إنشاؤهم أثناء الاختبار
- **المريض:** استخدام صفحة الحجز أونلاين

## 🧪 المرحلة 1: اختبار السوبر أدمن

### خطوات الاختبار

#### 1. تسجيل الدخول
1. افتح المتصفح على `http://localhost:3000/login`
2. أدخل البريد: `admin@platform.com`
3. أدخل كلمة المرور: `Admin@1234`
4. انقر على "تسجيل الدخول"
5. ✅ التحقق من التوجيه للوحة التحكم

#### 2. اختبار Dashboard
1. تحقق من عرض الإحصائيات:
   - [ ] إجمالي العيادات
   - [ ] إجمالي المستخدمين
   - [ ] إيرادات الشهر
   - [ ] موافقات معلقة
2. تحقق من الإحصائيات الإضافية:
   - [ ] حالة النظام
   - [ ] معدل النمو
   - [ ] الأمان
3. تحقق من النشاط الأخير:
   - [ ] عرض النشاطات
   - [ ] الطوابع الزمنية

#### 3. اختبار التنقل
1. حاول التنقل بين الصفحات المختلفة
2. تحقق من عمل جميع الروابط
3. تحقق من عدم وجود أخطاء في Console

#### 4. المشاكل المكتشفة
- [ ] سجل أي أخطاء
- [ ] سجل أي أزرار لا تعمل
- [ ] سجل أي مشاكل في التنقل
