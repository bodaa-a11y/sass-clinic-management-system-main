## السيناريوهات المدعومة

### 🏥 عيادة طبيب واحد (single_clinic)

**الميزات:**
- الطبيب يمكنه إضافة موظف استقبال
- لا اختيار طبيب في حجز المواعيد (يتم تعيين الطبيب تلقائياً)
- Dashboard مخصص يعرض "عيادة طبيب واحد"
- صفحة إدارة تظهر "إدارة الاستقبال" فقط
- صفحة جدولة تظهر "جدول العمل" مباشرة دون اختيار طبيب
- Sidebar يعرض "إدارة الاستقبال" بدلاً من "إدارة العيادة"

**القيود:**
- لا يمكن إضافة أطباء آخرين
- لا يمكن إدارة التخصصات والخدمات
- لا يمكن الوصول إلى صفحة التقارير

---

### 🏥 عيادة متعددة أطباء (multi_clinic)

**الميزات:**
- إدارة كاملة للموظفين (أطباء + استقبال)
- اختيار الطبيب في حجز المواعيد
- Dashboard مخصص يعرض "عيادة متعددة أطباء"
- صفحة إدارة كاملة (موظفين + تخصصات + خدمات)
- صفحة جدولة مع اختيار الطبيب
- الوصول إلى صفحة التقارير

---

### 🏥 مركز طبي (medical_center)

**الميزات:**
- كل مميزات عيادة متعددة أطباء
- إضافة قسم "الأقسام" في صفحة الإدارة
- إدارة الأقسام والمواقع
- Dashboard مخصص يعرض "مركز طبي"

---

## الأدوار والصلاحيات

### 👨‍⚕️ Doctor (طبيب)

**في single_clinic:**
- صلاحيات: patients, appointments, medical_records, prescriptions, staff_management_limited
- يمكنه: إضافة receptionist، فحص المرضى، كتابة الوصفات
- لا يمكنه: إدارة الفواتير والمدفوعات

**في multi_clinic/medical_center:**
- صلاحيات: patients, appointments, medical_records, prescriptions
- يمكنه: فحص المرضى، كتابة الوصفات
- لا يمكنه: إدارة الموظفين، الفواتير

---

### 👩‍💼 Receptionist (استقبال)

**الصلاحيات:**
- patients: view, create, edit
- appointments: view, create, edit, delete
- invoices: view, create, edit
- payments: view, create, edit
- medical_records: view
- لا يمكن: تعديل السجلات الطبية، الوصول لصفحة الإدارة

---

### 👨‍💼 Clinic Admin (مدير عيادة)

**الصلاحيات:**
- صلاحيات كاملة على جميع الموديولات
- يمكنه: إضافة/حذف موظفين، إدارة التخصصات والخدمات، الوصول للتقارير

---

### 👑 Super Admin (مدير عام)

**الصلاحيات:**
- صلاحيات كاملة على جميع الموديولات
- يمكنه: إدارة جميع العيادات في النظام
